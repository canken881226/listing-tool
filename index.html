Listing 写作规则

A9 + COSMO + Rufus + ISE 四维标准

文案规则总览
✓
Title：110–148字符；L1词必须在前60字符；包含印花/版型/颜色；用 for [场景1], [场景2] and [场景3] 自然带入多场景；不用破折号；不堆砌；结尾加尺码范围
✓
Bullet 1（ISE钩子）："For [具体购买时刻的买家身份] — [具体恐惧/障碍] → [产品如何消解] + [孩子反应+家长情绪回报]"
✓
Bullet 2（设计×场景）：描述产品最视觉化的特征，连接具体使用场景，让读者"看见"孩子穿着它
✓
Bullet 3（面料，强制）："FABRIC & FIT: Crafted from [X]% [Fiber] / [Y]% [Fiber] — [拉伸/舒适说明]. [尺码建议]. [参见尺码图]."
✓
Bullet 4（场景画面）：用感官语言描绘 WHERE+WHEN+WHO+HOW CHILD LOOKS，结合2-3个场景，以视觉情绪收尾
✓
Bullet 5（护理+礼品）："EASY CARE & GIFTING: Machine wash cold..." + 限制说明 + 礼品包装场景
✓
Description：HTML格式，5个 <b>小标题</b><br> 分段：WHO IT'S FOR / FABRIC & FIT / CARE / GREAT FOR / NOT FOR
✓
Q&A：12-15条；Rufus优先主题；每条答案含具体可验证内容
✓
Search Terms：230-249字节；仅L3词；零标题重复；全小写无标点
✗
禁止：关键词堆砌 / best·top·premium·guaranteed / 中文字符 / 竞品品牌名 / 未核查直接上线
Layer 判断口诀
L1
看见 — 核心类目词，前60字符+Bullet必出现
L2
听懂 — 场景/需求词，用完整自然句表达，不能直接照抄词组
L3
藏好 — 同义词/别叫法，只放后台Search Terms
