Listing 写作规则

A9 + COSMO + Rufus + ISE 四维标准 · 全员统一执行

文案规则总览
✓
Title：≤200字符；同一词≤2次；L1词必须在前80字符；后段用1个L2可读场景词
✓
Bullet 1：For [具体身份/场景] → [产品解决什么] → [用户获得的感受]（ISE格式）
✓
Bullet 3：必须标注面料成分百分比（如95%涤纶/5%氨纶）+ 具体可验证数字
✓
Bullet 4：含ISE情绪词：购买前障碍 → 购买后回报（妈妈视角）
✓
Bullet 5：含尺码范围、礼品场景、「不适合」使用限制说明
✓
Description：小标题分段 — WHO IT'S FOR / FABRIC & FIT / CARE / OCCASIONS / NOT FOR
✓
Q&A：A类≥15条；B类≥10条；每条回答含可验证具体内容（数字/材质/操作步骤）
✓
Search Terms：≤249字节；零重复前台词；无标点；Layer 3专用
✓
Attributes：Seller Central后台所有字段填满——空字段=COSMO知识图谱断点
✗
禁止：关键词堆砌 / best·top·premium·100%保证 / AI输出未人工核查直接上线
Layer 判断口诀
L1
看见 — 核心类目词，去掉它Amazon不知道你卖什么。Title前段 + Bullet必出现
L2
听懂 — 场景/需求词，用完整自然句说出来，不能直接照抄词组
L3
藏好 — 同义词/别叫法，只放后台Search Terms，前台不出现
